Logiciel a installer: 

  1- NodeJs 

  2- MySQL 

  2- Postman  

    

Directives: 

  1 - Utiliser le script .sql pour creer la base des donnees 

  2  - Faire un extract du zip file 

  3 - Ouvrir le projet avec Visual studio code. 

   

  4 - Avec Node package manager (npm) taper les commandes suivantes pour  installer les dependances necessaires. 

      Ouver un terminal et entrer les commandes qui suivent: 

     npm i -g express; //install la librairie express 

     npm i -g mysql;  //Install le mysql driver qui permet de se connecter a MySQL 

     npm i -g joi;  //Install Joi qui permet de faire la validation. 

     npm i -g nodemon; //Install node monitor to run the application. 

  
  5 - Vous etes bons pour commencer a tester le demos :) (Utiliser postman pour envoyer les HTTP request )
